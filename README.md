## 🌀 Philosophy
Inspired by the Japanese concept of **Danshari** (refusing, disposing, and separating), the Tamago Timer is designed to remove digital noise. It provides a clean, high-contrast interface that allows for deep work without the clutter of traditional productivity apps.

## 🛠️ Technical Architecture
This project serves as a showcase of minimalist UI systems and efficient state management.
* **Engine:** Built with **React** and **Vite** for near-instant load times.
* **Styling:** Utilizes **Tailwind CSS** to maintain a strict, high-contrast visual hierarchy.
* **Logic:** Implements precise interval tracking to ensure zero-drift focus sessions.

## ✨ Key Features
* **Minimalist Interface:** Zero distractions, focusing purely on the passage of time.
* **High-Contrast UI:** Optimized for both deep-focus and low-light environments.
* **Responsive Design:** Fully functional across desktop and mobile browsers.

## 🚀 Deployment
The Tamago Timer is deployed via **Vercel** as part of a decoupled digital infrastructure. 

### Local Development
1. Clone the repo: `git clone https://github.com/arturjesusp/tamago-timer.git`
2. Install dependencies: `npm install`
3. Start the engine: `npm run dev`
